#### Installation
Inside the application folder run:
```
export PYTHONPATH={FULL_PATH_TO_APPLICATION_FOLDER}
pip install -r requirements.txt
python setup.py bdist_wheel
pip install dist/news_service_lib-0.0.1-py3-none-any.whl
```
